# Sudoku-coursework88
Курсовая работа «игра Судоку» по дисциплине «Языки программирования» с графическим интерфейсом. Имеется генерация досок различной сложности (от Easy до Hard), сохранение доски в файл с возможностью в дальнейшем дорешать судоку, а также имеется отдельный компьютерный режим, где компьютер может решить указанное судоку.
